Pérpendiculaire d'un point

Utilisation: Model
Position: Pallette/Lignes
Bouton:

Description:

Permet de tracer des segments pérpendiculaire à une ligne, un arc ou une polyligne de base.
Sélectionnez l'élement qui vas servir de base dans un premier temp.
La fonction propose 3 options de tracé.

[Dynamique] [Limite] [Mesure]

La fonction Dynamique demande un point et trace une ligne dynamiquement entre ce point et la position du curseur.
Cliquez pour tracer la ligne.



La fonction Limite demande de sélectionner une limite (ligne, arc ou polyligne) puisun point de base.
Trace une ligne entre le point de base et la limite choisie.



La fonction Mesure trace une ligne depuis le point de base jusqu'a une mesure donnée.
La barre de commande précise les informations voulues a chaque étape.
Après avoir choisi la ligne de base, indiquer de quel coté de cette ligne tracer le segment, le point de base, puis la longueur souhaiter.
