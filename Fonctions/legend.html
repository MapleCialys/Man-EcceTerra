Légende

Description:

Permet de créer une légende automatiquement
Sélectionnez l'entièreté du dessin ou seulement les élements voulus, puis validez avec espace ou entrée.
La fonction vous dirige automatiquement vers la dernière présentation disponible.
Un cadre temporaire apparait pour indiquer de facon approximative l'éspace requis pour la légende.
A cette étape, il est possible de changer le nombre de colonne utiliser pour la légende.
Utilisez la touche "+" du pavé numérique pour augmenté le nombre de colonne et sur la touche "-" du pavé numérique pour réduire ce nombre.
Une fois la taille voulu sélectionnée, cliquez pour tracer la légende.
