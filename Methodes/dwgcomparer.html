DWG Comparer

La fonction DWGCOMPARER a pour but de comparer les éléments communs et différents entre deux fichiers DWG.
Ici, nous allons l'utiliser pour importer des extraction cadastrale dans les fichiers de calcul en évitant les doublons.

  1. Commençons par lancer la commande DWGCOMPARER


  2. Cette commande ouvre le panneau de comparaison de dessins.


  3. Utilisez le bouton pour choisir le fichier dwg résultant de l'extraction.

  4. Une fois le fichier chargé, affichez la page "Installation" du panneau de comparaison de dessins.


  5. Zoomez sur la zone a travailler et utiliser l'interrupteur a coté de "Pas dans le dessin courant" pour activer/désactiver la visualisation du fichier d'extraction (en vert).
    Malheureusement les projections et les distances sur lesquelles nous travaillons ne nous permettes pas d'avoir la précision souhaitée quand aux extractions. La comparaison ne vas donc pas bien détecter les lignes déjà présentes dans notre fichier dû au micro-décalage de distance ou d'orientation.
    C'est pour cela qu'il est important de vérifier quelles informations sont déjà présentes ou non.


  6. Une fois les éléments voulus repérés, sélectionnez les. Vous pouvez à tout moment activer ou désactiver la visualisation des objets a importer. Une fois l'ensemble des éléments sélectionnés, validez la sélection avec le bouton "Conserver".
    La sélection deviens maintenant grisâtre, les éléments ont bien étés importés.
    Note: Le bouton conserver n'est pas forcément collé au panneau de comparaison, mais la barre d'outils est obligatoirement quelque pars sur l'écran, généralement le coin supérieur gauche.



  7. Vous pouvez maintenant utiliser le bouton pour terminer l'opération.
