Courbes de niveau

Le traitement des courbes de niveau se déroule à partir du contrôleur de courbes de niveaux.
Seul les polylignes sur les calques E-MOD-COURB-MAITRESS et E-MOD-COURB-SECONDAI seront considérées comme des courbes de niveaux.

  • Permet de convertir les polylignes 3D (élévations différentes pour chaques sommets)
    en polylignes 2D (même élévation sur tous les sommets)

  • Permet de mettre rapidement des polylignes sur les calques des courbes de niveaux

  • Permet d'extraine les courbes de niveaux d'une surface TIN

  • Permet de gérer les étiquettes d'élévation:
    • Place les étiquettes au milieu de chaque
    • Place les étiquettes en fonction d'une ligne tracé
    • Place une étiquette sur une ligne voulue
    • Supprime toutes les étiquettes présentes

  • Gère l'intervalle entre chaque courbe maitresse

  • Gère l'intervalle entre chaque courbe à afficher

Avant d'extraire les courbes d'une surface TIN, il est conseillé de configurer la TIN avec des intervales de 0.1 pour avoir accès a toute la precision possible.
Cette option est disponible dans le panneau de propriétés après avoir sélectionner la TIN.
